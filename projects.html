Projects

CamScrabble (Spring 2016) Team Size:3
Programming Language: C, Python, C++
Designing and Implementing an automated scrabble game on Da Vinci Board using neutral networks and corner detection algorithm. For detection of grid coordinates algorithm has been designed which uses color filtering and morphological processing. Optical character recognition is done using neutral network. The aim of the project is to find all the new words formed on board and calculate total score for each user.

Relevance Filtering and Diverse Selection of Images (Summer 2015) Team Size:2
Programming Language: Matlab, Python
Designed a system which finds the relevant and diverse images present over social media based upon the given topic of interest. Selection was done based on visual features, textual features and flicker rank, by using KNN and clustering algorithm.
